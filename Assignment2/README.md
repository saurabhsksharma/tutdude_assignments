### [Task 1: Check if a Number is Even or Odd](./Task2.1.py)
Problem Statement:  Write a Python program that:
1. 	Takes an integer input from the user.

```
Enter a number: 7
7 is an odd number

Enter a number: 12
12 is an even number
```
